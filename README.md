# NewsFlow 📰

iOS news reading app built with UIKit and VIPER architecture.

## Tech Stack
- Swift
- UIKit + Auto Layout (programmatically)
- VIPER Architecture
- Alamofire
- Kingfisher
- NewsAPI

## Screens
- Home — top news feed with categories
- Detail — full article view
- Search — search by keyword
- Saved — bookmarked articles

## Installation
1. Clone the repo
2. Run `pod install`
3. Open `NewsFlow.xcworkspace`
4. Add your API key from newsapi.org
